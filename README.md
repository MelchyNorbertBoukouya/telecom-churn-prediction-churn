# Telecom Customer Churn Prediction 📊

## Project Overview
This project analyzes customer churn in the telecommunications sector using Machine Learning.

The objective is to identify customers likely to leave the company and help improve customer retention strategies.

## Dataset
IBM Telco Customer Churn Dataset (Kaggle)

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Machine Learning Modeling
5. Business Insights

## Models Used
- Logistic Regression

## Key Findings
- Monthly contract customers churn more
- High monthly charges increase churn risk
- New customers leave faster
- Long-term contracts improve loyalty

## Business Recommendation
Telecom companies should target monthly customers with loyalty offers and improve onboarding for new clients.
